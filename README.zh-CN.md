# 口播大纲动效剪辑 Skill

这个 skill 用来帮助 Codex 在 ChatCut 里剪辑中文口播视频。

它主要解决四件事：

1. 按口播内容添加大纲节点、标题节点、关键词弹出和 MG 动画。
2. 节点必须跟说话时间对齐，说到哪个词，哪个词再弹出。
3. 如果视频没有字幕，先补字幕，再安排 MG 位置，避免挡字幕。
4. 可以制作封面图，并把封面作为视频第一帧或开头静帧。

## 为什么 SKILL.md 用英文

`SKILL.md` 主要是给 Codex 执行时读取的。英文指令通常更容易保持稳定、精确，尤其是涉及工具调用、时间线、字幕、Motion Graphics 这些流程时。

但你不需要用英文理解它，所以这个仓库保留中文说明文件，方便你检查这个 skill 到底会做什么。

## 使用场景

你可以这样说：

```text
帮我分析这个口播视频，按说话内容添加大纲/标题节点和MG动画；如果没有字幕，先补字幕。
```

也可以这样说：

```text
帮我给这个视频做封面，系列是表达力觉醒，第12期，封面作为第一帧。
```

## 封面规则

制作封面前，Codex 必须先问：

- 这是哪个系列的视频？
- 这是第几期？

目前已有系列：

```text
表达力觉醒
```

这个系列的封面顶部固定写：

```text
🔥表达力觉醒 第x期
```

其中 `x` 替换成用户提供的期数。这个格式不要改成其他写法。

封面底部使用每条视频自己的封面标题，标题要更适合点击，可以比帖子标题更短、更直接。

## 默认剪辑原则

- 节点跟口播内容对齐，不按固定时间一次性弹出。
- 罗列多个概念时，每个概念分别查 transcript 时间，逐个出现。
- 方法论节点要保留到最后一个子点说完再消失。
- 有对比时，正向内容用 `✅`，反向内容用 `❌`。
- MG 避开人脸、嘴部、重要手势和字幕。
- 默认输出是可预览的 ChatCut 项目，不主动导出 MP4。
